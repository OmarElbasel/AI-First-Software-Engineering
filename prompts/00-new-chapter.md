# Generate New Chapter

Your task is to create ONE production-grade chapter for the AI-First Software Engineering Handbook.

Before writing:

1. Read CLAUDE.md
2. Read AGENTS.md
3. Read every file inside /meta
4. Read previous related chapters
5. Understand the curriculum position

Never write a chapter in isolation.

The handbook must remain internally consistent.

---

## Workflow

Step 1

Understand the objective of this chapter.

Step 2

Determine prerequisites.

Step 3

Generate a detailed outline.

Step 4

Review the outline.

Step 5

Generate the chapter.

Step 6

Review the chapter.

Step 7

Improve weak sections.

Only then finish.

---

Folder Structure, and Implementation with a single Real-World Scenario, per the
Constitution's v1.1 amendment. All other sections remain mandatory.

## Writing Rules

Always explain WHY before HOW.

Always compare alternatives.

Always explain trade-offs.

Always think like a Senior Software Architect.

Never assume the reader knows the topic.

Never explain syntax without context.

Never copy documentation.

Never generate filler.
